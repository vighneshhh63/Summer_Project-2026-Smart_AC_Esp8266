# Summer_Project-2026-Smart_AC_Esp8266
 ☀️ Summer Project: Alexa Voice-Controlled Smart AC (ESP8266)  During my summer break, instead of just scrolling through social media , I decided to build something practical. I converted my old, non-smart Panasonic AC into a voice-controlled Smart AC using a NodeMCU (ESP8266), an IR Transmitter, and Sinric Pro connected with Amazon Alexa.
